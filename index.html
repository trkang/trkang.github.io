PROJECTS

Web Crawler

Implemented a full crawler that uses a spacetime cache server to receive requests. The crawler creates a frontier and worker(s). When started created workers pick up an undownloaded link from the frontier, download it from the cache server, and then pass it into a scraper. Links received by the scraper are added to a list of undownloaded links in the frontier and the downloaded url is marked complete. Cycles until there are no more urls to be downloaded in the frontier. Uses tf-idf and cosine similarity to measure importance of tokens in text.
